# Kanban Board Explanation

## What is a Kanban Board?

A Kanban board is a visual project management tool used to track tasks as they move through different stages of work.

It uses columns such as:
- To Do
- In Progress
- Done

Each task is represented as a card that moves across the board.

---

## How Our Kanban Board Works

### Workflow Visualization

The Kanban board visually represents the progress of tasks:

- To Do: Tasks that have not started
- In Progress: Tasks currently being worked on
- Testing: Tasks under review
- Done: Completed tasks
- Blocked: Tasks that cannot proceed

---

### Work-In-Progress (WIP) Limits

To prevent overload, the system limits the number of tasks in progress:

- Maximum 3 tasks in "In Progress"
- Helps maintain focus and avoid bottlenecks

---

### Agile Support

The Kanban board supports Agile principles by:

- Allowing continuous delivery of features
- Providing flexibility to update tasks
- Improving visibility of progress
- Supporting iterative development

---

## Customization Choices

The following columns were added:

- Testing → ensures quality control before completion
- Blocked → identifies issues that delay progress

These additions improve workflow clarity and align with real-world software development practices.
