# Warehouse IT Asset Tracking System

## Project Description
The Warehouse IT Asset Tracking System is a software application designed to manage and monitor IT equipment within a warehouse environment. The system will allow IT administrators to track devices such as scanners, printers, laptops, monitors, and mobile printers used across different packing tables and operational areas.

The goal of the system is to improve visibility, reduce equipment loss, and simplify asset maintenance and reporting.

This system will allow administrators to:
- Register new IT assets
- Track asset location and status
- Record maintenance and repairs
- View asset history
- Generate reports

The system will be designed as a web-based application with a backend service and database.
